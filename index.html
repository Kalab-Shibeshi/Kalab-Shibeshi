Projects

Dwarf Frontier

Built with Python and Pygame, this game features:

  • Procedural terrain with biomes
  • AI dwarves that mine, explore, and react to the world
  • Physics for cave-ins and support structures
  • Weather systems like rain and snow
  • Zoomable ASCII art rendering
All coded by me, no copy-paste tutorials.
